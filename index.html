About

E-mail generator:

The application is under development, functionality and features are constantly being improved.

The steps below will help you get to know the application and build the template.

1. Start to create template

Very simple, just drag & drop

2. Change image src & alt

Upload images to your server and change src

3. Set background color

Select block and change color

4. Change padding

Select block and change padding

5. Internal editor / change content

The internal editor is activated by double-click on the block

6. Change HTML block code

For developers possible to change the block code directly

7. Email Template Output & Testing Tool

Recommended services are used to test the template, or use yours

Notes:

Some email clients render template look different from what you may have designed and tested. Unfortunately, we can't do much about the way Outlook or other email clients display HTML email.

Web fonts only work in some email clients:

– Apple / iOS Mail
– Android (default mail client, not Gmail app)
– Outlook 2000 / Outlook.com app
– Thunderbird

Outlook has no support for border-radius, so your rounded corners will be straight.

In Outlooks lines appearing is caused by a bug rendering. What you see is actually the color set behind your content.
